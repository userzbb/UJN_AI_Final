# 《算界旅人》核心玩法与系统设计（草案）

本文件定义游戏的核心玩法框架与主要系统：人物数值、装备与背包、技能与等级、天赋与技术树、据点与基建，以及与技术主题的关联方式。

---

## 1. 人物与基础数值系统

### 1.1 角色类型

- **主角：调试者**
  - 玩家唯一可直接操控的核心角色。
  - 数值成长由「等级」「时代加成」「天赋树」共同决定。
- **同伴 / 召唤物（可选后期扩展）**
  - 通过特定剧情或天赋解锁，如“算法幻象”“算力分身”等。

### 1.2 基础属性

建议采用少而精的属性体系，便于数值掌控：

- **HP（耐久 / 生命值）**：归零即战斗失败。
- **EP（能量 / 计算力）**：释放技能与战技时消耗，可通过休息与道具恢复。
- **ATK（攻击）**：基础伤害数值，受武器与等级影响。
- **DEF（防御）**：减轻所受伤害，受防具与时代科技影响。
- **SPD（速度）**：影响移动速度与行动顺序（若采用半回合制战斗）。
- **FCS（专注 / 精度）**：影响暴击率、技能命中率以及特定解谜互动的成功率。

### 1.3 派生数值

- **CRIT（暴击率）**：由 FCS 与特定装备/天赋决定。
- **RESIST（抗性）**：针对“过热”“干扰”“噪声”等特殊状态的抗性。

## 2. 装备与背包系统

### 2.1 装备槽位

- 武器 x1（近战/远程/施法媒介）
- 头部防具 x1（头盔、护目镜等）
- 身体防具 x1（护甲、围裙等）
- 饰品 x2（戒指、吊坠、怀表等）

### 2.2 装备来源与时代绑定

- 每个时代解锁一批对应技术的装备：
  - 机械时代：齿轮类、算盘类、差分机部件。
  - 电子管时代：高压电路、真空管放大器等。
  - 集成电路时代：晶体管阵列、集成控制模块等。
- 高时代装备可向下通用，但可能在低时代受到“兼容性惩罚”（例如效果打折）。

### 2.3 背包与负重

- **格子背包 + 负重上限**：
  - 背包按格子计数，道具/装备占用不同格数。
  - 负重超过上限会降低 SPD 或增加技能冷却时间。
- 特殊装备（如「机匠围裙」「扩展模组背包」）可提升负重上限。

### 2.4 装备品质与强化

- 品质等级：普通 → 精良 → 稀有 → 传承 → 时代奇迹。
- 强化方式：
  - 使用对应时代的「工程材料」进行强化（齿轮碎片、真空管残件、晶体屑等）。
  - 强化提高基础数值，部分稀有装备在特定强化等级解锁新被动效果。

## 3. 技能、等级与经验系统

### 3.1 等级与经验

- 角色通过以下方式获得 EXP：
  - 战斗胜利。
  - 完成任务（主线/支线/时代事件）。
  - 探索隐藏技术节点（阅读碑文、修复古老设备）。
- 升级收益：
  - 提升基础属性（HP、ATK、DEF 等）。
  - 获得「技能点」与「天赋点」。

### 3.2 技能分类

- **战斗类技能**：
  - 近战技（齿轮连击、继电冲击、过载斩击）。
  - 远程技（数据射线、电弧链等）。
- **工具类技能**：
  - 解谜、机关、互动（解析之眼强化版、远程操作机械臂、重写访问权限等）。
- **时代特化技能**：
  - 每个时代解锁 1~2 个与当代技术强关联的标志性技能。

### 3.3 技能习得方式

- 通过升级获得的技能点在「技能树」中解锁。
- 部分技能由特定 NPC 教授（如艾达教指令序列相关技能）。
- 少量终极技能需完成时代的大型事件链后解锁。

## 4. 天赋与技术树

### 4.1 天赋树结构

建议采用「三大主系 + 若干分支」：

- **逻辑系（Logic）**：
  - 提升 FCS、技能命中、解谜能力。
  - 技能例：
    - 精确推理：减少技能冷却偏差。
    - 严格证明：对“错误型敌人”（投错公式、异常数据）伤害提高。

- **工程系（Engineering）**：
  - 提升 HP、DEF、负重与环境互动能力。
  - 技能例：
    - 加固结构：受到环境伤害时减免。
    - 快速修复：战中临时恢复装备耐久或召唤简易掩体。

- **算力系（Compute）**：
  - 提升 EP、技能威力、召唤与复制能力。
  - 技能例：
    - 并行思维：一定时间内技能可“同时存在多个实例”。

### 4.2 时代技术树

- 每个时代有一小段「时代技术树」，玩家通过完成相关任务点亮节点：
  - 机械时代：差分机 → 分析机 → 穿孔卡片控制。
  - 电子管时代：放大 → 振荡 → 逻辑门电路。
  - ……
- 时代技术树带来的是：
  - 解锁新装备配方。
  - 部分全局被动加成（如提升相关类型技能效果）。

## 5. 据点与基建系统

### 5.1 初始据点：「中立调试站」

- 位置：时间编译器所在的中立空间中，作为跨时代 HUB。
- 功能模块：
  - 休息与回复：恢复 HP/EP，推进部分角色对话。
  - 仓库与装备管理：存放不常用装备与材料。
  - 研究室：查看与升级时代技术树。

### 5.2 时代前哨与城市基建

- 每个时代的关键城市中，玩家可以逐步解锁功能建筑：
  - **工坊**：装备打造与强化。
  - **实验台**：进行技能/天赋重置、测试新组合。
  - **资料馆**：解锁背景文档、技术图谱，并提供与结局相关的隐藏线索。
- 基建升级：
  - 使用「时代资源」与任务奖励提升建筑等级。
  - 高等级工坊可制作更高级别的装备。

### 5.3 与结局的关系

- 据点的整体发展水平（如各建筑等级之和）将作为评估玩家“对技术的整体治理能力”的隐藏参数之一：
  - 只堆战斗力、忽略资料馆与研究室，可能朝 BAD_END / NORMAL_END 方向偏移。
  - 平衡战斗、研究与资料整理，利于解锁 TRUE_END。


- 为每一类系统补充「具体样例」：
  - 至少设计 5~10 件每个时代的代表性装备条目。
  - 设计 10~20 个基础技能，并标明所属系别/时代。
  - 定义 1 版等级曲线（每级所需 EXP、属性成长）。
- 与现有剧情文档联动：
  - 为 `ERA_MECHANICAL_COMPUTING.md` 中出现的 NPC 和事件挂接具体奖励（装备/技能/天赋点）。
