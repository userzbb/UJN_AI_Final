# 时栈 (TimeStack: A Computing History TUI)

## 世界观 (Worldview)

在名为 **「时栈 (The TimeStack)」** 的世界中，历史被一层层“压栈”在不同的计算纪元里——
从齿轮咬合的机械工坊，到真空管闪烁的机房，再到晶体管密布的硅片平原。

你是一名被时间选中的 **「调试者 (Debugger)」**。
某种未知的错误让这些时代之间的调用关系出现紊乱：本应依次演进的技术，被打乱、折叠，甚至相互覆盖。
你的任务不是改变历史，而是 **修复计算机发展过程中被破坏的关键节点**，让每一层时代按照应有的顺序入栈与出栈。

## 旅程 (The Journey)

你的冒险将是一部可交互的 **计算机发展简史**：

1.  **机械计算时代**：齿轮与蒸汽的轰鸣，差分机的精密运转。
2.  **电子管时代**：真空中的微光，巨型机房的诞生。
3.  **集成电路时代**：微缩的奇迹，硅谷的崛起。
4.  **网络与算力时代**：万物互联与高密度算力中心。

每个时代都有自己的城市、人物与技术矛盾。通过探索、对话与战斗，你将逐步获得承载技术成果的 **「时代信标 (Epoch Token)」**，用以解锁通往下一时代的传送节点。

## 人物设定 (Character)

-   **代号**: 调试者 (Debugger)
-   **身份**: 在各个计算时代之间“跳转”的技术观察者与修补者。
-   **能力**:
    -   **Vim-Fu**: 使用类 Helix/Vim 的键位，在终端世界中移动与战斗。
    -   **解析之眼 (Parser Eye)**: 看见被隐藏的技术节点、协议残片与错误日志。
    -   **代码注入 (Code Injection)**: 在特定位置修改环境参数，影响关卡结构或敌人行为。

## 交互方式 (Controls - Helix Mode)

本游戏采用 **Helix/Vim** 风格的模态编辑操作，键盘是你最重要的“武器与工具”：

### Normal Mode (移动与探索)
-   `h` `j` `k` `l`: 左 下 上 右 移动
-   `Space`: 打开 **操作菜单 (Action Menu)**
    -   `e`: 装备/图书 (Equipment/Lore)
    -   `s`: 存档/读档 (Save/Load)
-   `:`: 进入命令模式

### Command Mode (指令与系统)
-   `:w`: 保存当前进度
-   `:q`: 退出游戏
-   `:new`: 开始新游戏
